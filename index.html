SquadBets

This repository is configured to deploy a static web build into the docs/ folder.

If you are deploying with GitHub Actions (recommended):

  1. Go to your repository Settings → Pages.
  2. Under “Build and deployment”, choose GitHub Actions.
  3. Push to main (or run the “Deploy to GitHub Pages” workflow).

If you are deploying from a branch:

  1. Set Pages Source = main, Folder = /docs.
  2. Run npm run build:web locally and commit the generated docs/ output.

This is a placeholder index.html to avoid “File not found”. The real app is generated by Expo export.
